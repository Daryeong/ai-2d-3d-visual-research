Project Overview

AI를 이미지 생성 도구가 아닌 브랜드 경험 탐색 도구로 사용했습니다.

Core Question

같은 프롬프트를 여러 AI 도구에 입력했을 때, 각 도구는 미래적이고 초현실적인 아이웨어 브랜드의 세계관을 어떻게 다르게 해석하는가?

Application Point

결과물을 나열하는 데 그치지 않고 실패 결과와 3D 확장 가능성까지 비교해 리테일 공간, 아이웨어 오브젝트, 디지털 아카이브 관점으로 정리했습니다.

Shared Prompt System

세 가지 프롬프트로 캠페인, 공간, 그래픽 시스템을 비교했습니다.

Prompt 01

Fashion Campaign Visual

Futuristic eyewear fashion campaign image, dark showroom, reflective sunglasses, chrome styling, surreal luxury mood, cinematic lighting, minimal composition.

Prompt 02

Retail Space Mood

Surreal futuristic retail space for a luxury eyewear brand, mirror walls, chrome objects, transparent lens sculptures, dark minimal atmosphere, immersive showroom mood.

Prompt 03

Abstract Brand Graphic System

Abstract visual graphic system for a futuristic eyewear fashion brand, distorted lens shapes, chrome reflections, transparent glass textures, dark gradient background, minimal poster composition.

Experiment 01

2D Visual Comparison

Canva, Codex, Firefly, Flow, Grok, MiriCanvas, Nano Banana 2 결과를 동일 프롬프트 기준으로 비교했습니다.

3D Prompt System

3D 오브젝트 실험은 리테일 조형물, 아이웨어 설치물, 제품 진열 구조로 나누어 비교했습니다.

Prompt 01

Robotic Retail Object

Robotic retail display object for an eyewear brand, combining eye motifs, polished chrome metal, glossy lens surfaces, mechanical joints, and sculptural futuristic form.

Prompt 02

Mirror Lens Robotic Installation

Futuristic showroom sculpture combining mirror, lens, chrome metal, transparent acrylic, and biomorphic form.

Prompt 03

Kinetic Robotic Eyewear Display

Kinetic robotic retail display structure with a rotating circular frame, floating lens holders, chrome mechanical arms, and a transparent acrylic base.

Experiment 02

3D Object Exploration

2D 이미지에서 나온 브랜드 무드를 초현실적 오브젝트, 제품 디스플레이, 디지털 리테일 경험으로 확장할 수 있는지 확인했습니다.
