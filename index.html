Core Question

같은 프롬프트를 여러 AI 도구에 입력했을 때, 각 도구는 미래적, 초현실적, 럭셔리한 아이웨어 브랜드 세계관을 어떻게 다르게 해석하는가?

Application Point

결과물을 예쁘게 나열하는 데서 끝내지 않고, 실패 결과와 3D 확장 가능성까지 비교해 리테일 공간, 쇼윈도 오브젝트, 디지털 아카이브 관점으로 정리했습니다.

Shared Prompt System

2D 비주얼 실험은 세 가지 프롬프트로 캠페인, 공간, 그래픽 시스템을 비교했습니다.

Prompt 01

Fashion Campaign Visual

Futuristic eyewear fashion campaign image, dark showroom, reflective sunglasses, chrome styling, surreal luxury mood, cinematic lighting, minimal composition.

Prompt 02

Retail Space Mood

Surreal futuristic retail space for a luxury eyewear brand, mirror walls, chrome objects, transparent lens sculptures, dark minimal atmosphere, immersive showroom mood.

Prompt 03

Abstract Brand Graphic System

Abstract visual graphic system for a futuristic eyewear fashion brand, distorted lens shapes, chrome reflections, transparent glass textures, dark gradient background, minimal poster composition.

Experiment 01

2D Visual Comparison

Canva, Codex, Firefly, Grok, GPT Image 2, Nano Banana 2 결과를 동일 프롬프트 기준으로 비교했습니다.

3D Prompt System

3D 오브젝트 실험은 리테일 오브제, 쇼윈도 설치물, 제품 진열 구조로 나누어 비교했습니다.

Prompt 01

Robotic Retail Object

Robotic retail display object for an eyewear brand, combining eye motifs, polished chrome metal, glossy lens surfaces, mechanical joints, and sculptural futuristic form. Designed as a strange but luxurious showroom object for an experimental fashion retail space.

Prompt 02

Mirror Lens Robotic Installation

Futuristic showroom sculpture combining mirror, lens, chrome metal, transparent acrylic, and biomorphic form. A surreal installation object for a luxury sunglasses brand, suitable for a window display or digital retail space.

Prompt 03

Kinetic Robotic Eyewear Display Structure

Kinetic robotic retail display structure for a futuristic eyewear brand, rotating circular frame, floating lens holders, chrome mechanical arms, transparent acrylic base, and minimal luxury showroom design. Designed to display sunglasses in an immersive fashion retail space.

Experiment 02

3D Object Exploration

Copilot 3D, Tripo AI, Womp, Tencent Hunyuan3D, Codex 결과를 동일 프롬프트 기준으로 비교했습니다. front / side / back 순서로 정리했습니다.

Insights for Gentle Monster

AI 실험을 리테일, 공간, 콘텐츠 경험으로 연결합니다.

01

AI를 브랜드 컨셉 탐색 도구로 사용

02

2D에서 3D로 확장

결과 이미지를 실제 전시와 리테일 오브젝트로 연결할 수 있는지 검토했습니다.

03

실패 결과까지 분석

왜곡과 브랜드 무드 불일치를 다음 프롬프트 조정 기준으로 정리했습니다.

04

웹 경험으로 구조화
